# Template

AGENTS.md template for new directories and modules. Only include sections with content.

## Minimal

```markdown
## AGENTS.md -- <name>

<One-line description.>
```

## Standard

```markdown
## AGENTS.md -- <name>

<One-line description.> Tier <N>.

### <Unique section>

<What is specific to this directory — not repeated from root.>
```

## Module

For code packages and project-specific knowledge bases.

```markdown
## AGENTS.md -- <module>

<One-line technical description.>

### API

| File | Purpose |
|------|---------|

### Rules

1. <Constraint>

### Dependencies

- `<dep>` — <role>
```
