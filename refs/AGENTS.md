## AGENTS.md -- refs

Verified external knowledge. Tier 3.

Authoritative knowledge from specs, official docs, peer-reviewed papers.
refs = external truths; notes = internal insights.
Unverified knowledge belongs in `notes/`, not here.

### Verification

1. Identify primary source
2. Verify claim against source
3. Record URL or citation
4. Mark date

### Format

```
# <Topic>

Source: <URL or citation>
Verified: <YYYY-MM-DD>

<Content>
```
